# Tuxruku ASA Manager (TAM) 🐧🦖

**The Linux Node Command Center for Ark: Survival Ascended.**

Tuxruku ASA Manager is a lightweight, open-source, and extremely powerful Web GUI for managing Ark: Survival Ascended servers on Linux. Built with Node.js, it avoids the heavy overhead of traditional control panels and gives you exactly what you need.

##  Features
* **Multi-Server Management:** Control multiple instances from a single dashboard.
* **Cron Automation:** Schedule automated backups, wipes, and restarts.
* **Dynamic Settings Engine:** Edit `GameUserSettings.ini` and `Game.ini` through a visual UI.
* **Live RCON Dashboard:** Chat with players, view live logs, and execute commands.
* **CurseForge Integration:** Search and install mods directly from the web panel.
* **Multilingual (i18n):** Fully supports English and Norwegian (more coming soon!).
* **Self-Healing:** Runs beautifully on PM2 for zero-downtime management.

## 🚀 Installation (Linux)

**1. Download and run the installer script as root:**

wget https://raw.githubusercontent.com/ITzHandal/Tuxruku/main/install.sh

sudo bash install.sh


**2. Follow the on-screen instructions!**
